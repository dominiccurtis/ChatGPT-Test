Dominic CURTIS : KoinKeeper - Versions

Created by Dominic CURTIS, last modified on 14.03.2023

Version 1.0.0 (Released on January 1, 2020): Initial release of KoinKeeper, featuring support for managing Bitcoin and Ethereum wallets.


Version 1.1.0 (Released on March 15, 2020): Added support for managing Litecoin and Ripple wallets, as well as a new user interface for easier navigation.


Version 1.2.0 (Released on June 30, 2020): Introduced a new feature for managing recurring transactions, as well as enhancements to the security protocols.


Version 2.0.0 (Released on December 1, 2020): Major release of KoinKeeper, featuring a redesigned user interface, new support for managing additional cryptocurrencies, and integration with third-party exchanges.


Version 2.1.0 (Released on March 1, 2021): Added support for managing NFTs (non-fungible tokens), as well as improvements to the performance and stability of the application.
