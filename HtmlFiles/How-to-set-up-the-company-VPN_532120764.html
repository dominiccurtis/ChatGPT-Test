Dominic CURTIS : How to set up the company VPN

Created by Dominic CURTIS on 09.03.2023
  • Download "My Imaginary VPN" from www.myimaginaryvpn.com


  • Double-click the .exe file and install the software on your computer


  • Run the software, and during the initial setup, enter Port number "1060", username: "ImaginaryAdmin", and password: "ImaginaryPassword".


  • In the taskbar, right-click on the icon for "My Imaginary VPN" (a unicorn), and select the option "Activate".


  • If you ever need to turn off the VPN, right-click on the icon and select "Deactivate".
